================================================================================

STEP 4: CONFIGURE ON SERVER
===========================

VPS Hosting:
-----------
1. SSH to your server
2. Extract: tar -xzf pos-system-deploy.tar.gz
3. Install Node.js (see DEPLOY_TO_HOSTINGER.md)
4. Install dependencies: cd backend && npm install
5. Start with PM2: pm2 start server.js
6. Configure Nginx (see guide)

Cloud Hosting (hPanel):
----------------------
1. Go to Node.js section in hPanel
2. Create new Node.js app
3. Upload files
4. Set startup: backend/server.js
5. Start application
