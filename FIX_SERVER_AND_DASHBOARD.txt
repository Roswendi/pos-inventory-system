TROUBLESHOOTING:
================

IF "Cannot GET /" ERROR:
------------------------

IF DASHBOARD SHOWS "NO DATA":
-----------------------------
‚Üí Server might not be running
‚Üí API might be failing

--------------------------
  lsof -ti:5001 | xargs kill
  Then start server again

IF BROWSER CAN'T CONNECT:
-------------------------
1. Check server is running (lsof -ti:5001)
2. Check firewall settings
3. Try: http://127.0.0.1:5001 instead
