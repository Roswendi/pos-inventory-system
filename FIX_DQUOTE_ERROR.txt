================================================================================
    HOW TO FIX "DQUOTE" ERROR
================================================================================

WHAT IS "DQUOTE"?
-----------------
"dquote" = "double quote" 
The terminal is waiting for you to close a quote that you opened.

This happens when:
- You type an opening quote " but forget the closing quote "
- You use the wrong type of quote (smart quotes vs straight quotes)
- You press Enter before closing the quote

================================================================================
HOW TO FIX IT
================================================================================

METHOD 1: Close the Quote (If You See "dquote>")
--------------------------------------------------
If you see:
  dquote>

Just type a closing quote:
  "

Then press Enter.

METHOD 2: Cancel and Start Over
---------------------------------
Press: Ctrl + C
This cancels the current command and starts fresh.

Then type the command again with proper quotes.

METHOD 3: Use the Correct Quote Type
-------------------------------------
Use STRAIGHT quotes (not curly/smart quotes):

CORRECT:  "text"  (straight quotes)
WRONG:    "text"  (curly/smart quotes)

1. Always type BOTH quotes:
