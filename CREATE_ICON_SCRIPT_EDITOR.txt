ALTERNATIVE: TextEdit Method (If Script Editor Doesn't Work)
=============================================================

STEP 1: Open TextEdit
----------------------
1. Press: Cmd + Space
2. Type: "TextEdit"
3. Press: Enter

STEP 2: Create Script
---------------------
1. Make sure it's in "Plain Text" mode:
   - Format â†’ Make Plain Text (or Shift+Cmd+T)

2. Paste this:

  #!/bin/bash
  cd "/Users/endiroswendi/Documents/ Old Laptop Documents/CAO/FINANCIAL REPORT/pos-inventory-system/backend"
  if ! lsof -ti:5001 > /dev/null 2>&1; then
      osascript -e 'tell app "Terminal" to do script "cd '"'"'/Users/endiroswendi/Documents/ Old Laptop Documents/CAO/FINANCIAL REPORT/pos-inventory-system/backend'"'"' && node server.js"'
      sleep 5
  fi
  open "http://localhost:5001"

STEP 3: Save
------------
1. File â†’ Save (Cmd+S)
2. Name: "Start TDO POS.command"
   (IMPORTANT: Must end with .command)
3. Where: Desktop
4. Format: Plain Text
5. Click: "Save"

STEP 4: Make Executable
-----------------------
1. Open Terminal
2. Run:
   chmod +x ~/Desktop/"Start TDO POS.command"

STEP 5: Test
------------
Double-click "Start TDO POS.command" on desktop
It should launch the app!
