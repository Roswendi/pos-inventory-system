✅ EASY STEPS TO UPLOAD TO GITHUB
================================================================================

STEP 1: CREATE GITHUB REPOSITORY
=================================

1. Go to: https://github.com
2. Click: + (top right) → New repository
3. Name: pos-inventory-system
4. Choose: Public or Private
5. DO NOT check "Initialize with README"
6. Click: Create repository

================================================================================

STEP 2: RUN SETUP SCRIPT (EASIEST)
===================================

On your computer, run:

  bash SETUP_GITHUB.sh

This script will:
✓ Initialize Git
✓ Configure Git
✓ Add all files
✓ Create commit
✓ Connect to GitHub
✓ Push your code

Just follow the prompts!

================================================================================

STEP 3: MANUAL SETUP (If Script Doesn't Work)
==============================================

Open Terminal and run:

# Navigate to project
cd "/Users/endiroswendi/Documents/ Old Laptop Documents/CAO/FINANCIAL REPORT/pos-inventory-system"

# Initialize Git
git init

# Configure Git (first time only)
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# Add files
git add .

# Commit
git commit -m "Initial commit: POS Inventory System"

# Add GitHub remote (replace with your URL)
git remote add origin https://github.com/your-username/pos-inventory-system.git

# Push to GitHub
git branch -M main
git push -u origin main

================================================================================

STEP 4: AUTHENTICATION
======================

When pushing, GitHub will ask for credentials:

OPTION A: Personal Access Token (Recommended)
----------------------------------------------
1. GitHub → Settings → Developer settings
2. Personal access tokens → Tokens (classic)
3. Generate new token
4. Copy token
5. Use token as password when pushing

OPTION B: GitHub CLI
--------------------
Install: brew install gh
Login: gh auth login

================================================================================

STEP 5: VERIFY
==============

1. Go to your GitHub repository
2. You should see all your files
3. Done! ✅

================================================================================

DEPLOY FROM GITHUB TO HOSTINGER
================================

METHOD 1: Download from GitHub
-------------------------------
1. Go to your GitHub repository
2. Click: Code → Download ZIP
3. Extract and upload to Hostinger

METHOD 2: Clone on Server
--------------------------
If Hostinger has SSH access:

  git clone https://github.com/your-username/pos-inventory-system.git
  cd pos-inventory-system
  # Follow deployment instructions

METHOD 3: Use Hostinger's GitHub Integration
----------------------------------------------
1. In Hostinger hPanel
2. Look for "Git" or "GitHub" integration
3. Connect your repository
4. Auto-deploy on push

================================================================================

UPDATE YOUR CODE
================

After making changes:

1. Add changes:
   git add .

2. Commit:
   git commit -m "Description of changes"

3. Push:
   git push origin main

Your changes are now on GitHub!
