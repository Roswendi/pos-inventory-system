STEP 2: Check Backend is Running
---------------------------------
- Backend should be on: http://localhost:5001
- Check console for: "Server running on http://localhost:5001"

STEP 3: Test Outlets API
-------------------------
- Open browser console (F12)
- Go to Network tab
- Click on Outlets page
- Check if request to /api/outlets is successful

STEP 4: Check Data Directory
-----------------------------
- Verify: backend/data/outlets/outlets.json exists
- If not, it will be created automatically

================================================================================
TROUBLESHOOTING:
================================================================================

PROBLEM: Still getting "Error loading outlets"
-----------------------------------------------
SOLUTION:
1. Check backend server is running on port 5001
2. Check browser console (F12) for specific error
3. Check Network tab for failed API request
4. Verify backend/data/outlets/ directory exists
5. Restart backend server

PROBLEM: "Network Error" or "Failed to fetch"
----------------------------------------------
SOLUTION:
- Backend server is not running
- Start backend: cd backend && npm start
- Check port 5001 is not in use

PROBLEM: "401 Unauthorized"
---------------------------
SOLUTION:
- Make sure you're logged in
- Check token in localStorage
- Try logging out and logging back in

PROBLEM: "500 Internal Server Error"
-------------------------------------
SOLUTION:
- Check backend console for error details
- Verify data/outlets directory exists
- Check file permissions
