================================================================================
    HOW TO FIX MISSING ICON IN BROWSER
================================================================================

If you don't see the icon, it's usually because:
1. Icons are not in frontend/public/ folder
2. App wasn't rebuilt after adding icons
3. Browser cache needs clearing

================================================================================
STEP 1: CHECK IF ICONS EXIST
================================================================================

Run this command:
  cd "/Users/endiroswendi/Documents/ Old Laptop Documents/CAO/FINANCIAL REPORT/pos-inventory-system/frontend/public"
  ls -la logo*.png

You should see:
  - logo192.png
  - logo512.png

If you DON'T see them, you need to copy your TDO icons there first!

================================================================================
STEP 2: REBUILD THE APP (IMPORTANT!)
================================================================================

After adding icons, you MUST rebuild the app:

1. Stop the server (if running):
   - Go to terminal where server is running
   - Press Ctrl+C

2. Rebuild:
   cd "/Users/endiroswendi/Documents/ Old Laptop Documents/CAO/FINANCIAL REPORT/pos-inventory-system/frontend"
   npm run build

3. Wait for "Build successful"

4. Start server again:
   npx serve -s build -p 3000

5. Refresh browser (Cmd+Shift+R or Ctrl+Shift+R)

================================================================================
STEP 3: CLEAR BROWSER CACHE
================================================================================

Chrome:
1. Press Cmd+Shift+Delete (Mac) or Ctrl+Shift+Delete (Windows)
2. Select "Cached images and files"
3. Click "Clear data"
4. Refresh page (Cmd+Shift+R)

Safari:
1. Safari menu → Preferences → Advanced
2. Check "Show Develop menu"
3. Develop menu → Empty Caches
4. Refresh page (Cmd+Shift+R)

Firefox:
1. Press Cmd+Shift+Delete (Mac) or Ctrl+Shift+Delete (Windows)
2. Select "Cache"
3. Click "Clear Now"
4. Refresh page (Cmd+Shift+R)

================================================================================
STEP 4: HARD REFRESH
================================================================================

After clearing cache, do a hard refresh:

Mac:
  Cmd + Shift + R

Windows/Linux:
  Ctrl + Shift + R

This forces browser to reload everything.

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Icons don't exist in public folder
--------------------------------------------
SOLUTION:
- Copy your TDO icons (logo192.png, logo512.png) to frontend/public/
- See COPY_ICON_INSTRUCTIONS.txt for how to copy

PROBLEM: Icons exist but still not showing
-------------------------------------------
SOLUTION:
- Rebuild: npm run build (MUST do this after adding icons!)
- Clear browser cache
- Hard refresh: Cmd+Shift+R
- Check browser console (F12) for errors

PROBLEM: Icon shows as default/favicon
--------------------------------------
SOLUTION:
- Icons might be wrong size
- Check: logo192.png is 192x192, logo512.png is 512x512
- Rebuild after fixing

PROBLEM: Browser console shows 404 for icons
--------------------------------------------
SOLUTION:
- Icons not in build folder
- Rebuild: npm run build
- Check icons are in frontend/public/ before building

================================================================================
VERIFY IT WORKED
================================================================================

After fixing:
1. Browser tab shows your TDO icon (not default)
2. No errors in browser console (F12)
3. Icons load when you check Network tab
