4. Click Upload
5. Select all files from extracted folder
6. Upload

================================================================================

STEP 3: CONFIGURE ON SERVER
============================

1. SSH to server (or use Terminal in hPanel)
2. Run:
   cd public_html
   npm install --production
   cp .env.example .env
   nano .env

3. Add to .env:
   NODE_ENV=production
   PORT=5001
   JWT_SECRET=your-secret-key

4. Set permissions:
   chmod -R 755 .
   chmod -R 777 data
   chmod -R 777 uploads

5. Start:
   pm2 start server.js --name "pos-system"
