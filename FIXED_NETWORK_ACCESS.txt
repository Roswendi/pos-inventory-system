IF STILL NOT WORKING:
=====================

CHECK 1: Server Running?
-------------------------
  lsof -ti:5001

If empty, server is NOT running!
Start it!

CHECK 2: Same WiFi Network?
---------------------------
- Both devices must be on SAME WiFi
- Check WiFi name (SSID) is identical
- Cannot be on different networks

CHECK 3: Firewall (If Re-enabled)
----------------------------------
If you re-enabled firewall:
1. System Settings ‚Üí Network ‚Üí Firewall
2. Add Node.js exception
3. OR allow port 5001

CHECK 4: IP Address Changed?
----------------------------
If IP changed, find new IP:
  ifconfig | grep "inet " | grep -v 127.0.0.1

Use the NEW IP address!
