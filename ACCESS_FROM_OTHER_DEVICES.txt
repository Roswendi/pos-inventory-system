FIND YOUR IP ADDRESS (If Different):
====================================

If 192.168.0.200 doesn't work, find your actual IP:

ON MAC:
-------
  Open Terminal and run:
    ifconfig | grep "inet " | grep -v 127.0.0.1

  Look for something like: 192.168.x.x or 10.0.x.x

ON WINDOWS:
-----------
  Open Command Prompt and run:
    ipconfig

  Look for "IPv4 Address" under your WiFi adapter

ON LINUX:
---------
  Run:
    hostname -I

TROUBLESHOOTING:
================

PROBLEM: "Cannot reach" or "Connection refused"
------------------------------------------------
SOLUTION:
1. Make sure server is running on main computer
2. Check both devices are on SAME WiFi
3. Check firewall settings (see below)
4. Try: http://192.168.0.200:5001 (not localhost)

PROBLEM: "This site can't be reached"
-------------------------------------
SOLUTION:
1. Verify IP address is correct
2. Check server is running (look at Terminal)
3. Make sure port 5001 is not blocked

PROBLEM: Firewall Blocking
---------------------------
SOLUTION (macOS):
1. System Settings ‚Üí Network ‚Üí Firewall
2. Click "Options"
3. Make sure Node.js is allowed
4. Or add exception for port 5001

SOLUTION (Windows):
1. Windows Defender Firewall ‚Üí Allow an app
2. Add Node.js or port 5001

PROBLEM: Different WiFi Networks
---------------------------------
SOLUTION:
- Both devices MUST be on the same WiFi
- Cannot access from different networks
- For remote access, use cloud deployment (see guide)
