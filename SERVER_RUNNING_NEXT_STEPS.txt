================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Browser shows "Can't connect"
---------------------------------------
SOLUTION:
- Make sure server is still running (don't close terminal)
- Check URL is exactly: http://localhost:3000
- Try refreshing the page

PROBLEM: Icon doesn't show
---------------------------
SOLUTION:
- Check icons exist: ls frontend/public/logo*.png
- Rebuild: npm run build
- Clear browser cache
- Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)

PROBLEM: App shows errors
-------------------------
SOLUTION:
- Make sure backend is running on port 5001
- Check browser console (F12) for errors
- Verify API URL is correct
