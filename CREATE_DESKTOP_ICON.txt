================================================================================
    HOW TO CREATE DESKTOP ICON/SHORTCUT FOR YOUR APP
    Make Your POS App Accessible from Desktop
================================================================================

There are several ways to create a desktop icon on Mac:

================================================================================
METHOD 1: CREATE APPLE SCRIPT (Easiest for Mac)
================================================================================

STEP 1: Create Script
---------------------
1. Open "Script Editor" (Applications → Utilities → Script Editor)
2. Paste this code:

   tell application "Safari"
     activate
     open location "http://localhost:3000"
   end tell

   OR for Chrome:

   tell application "Google Chrome"
     activate
     open location "http://localhost:3000"
   end tell

3. File → Save
4. File Format: Application
5. Name it: "POS Inventory System"
6. Save to Desktop

STEP 2: Add Icon
----------------
1. Right-click the app on Desktop
2. Get Info
3. Click the icon in top left
4. Press Cmd+C to copy your TDO icon
5. Click the app icon again
6. Press Cmd+V to paste

STEP 3: Use It
--------------
- Double-click desktop icon
- Opens browser to your app
- Works like a native app!

================================================================================
METHOD 2: USE CHROME "CREATE SHORTCUT" (Easiest Overall)
================================================================================

STEP 1: Open Your App
---------------------
1. Open Chrome browser
2. Go to: http://localhost:3000
3. Make sure app is working

STEP 2: Create Desktop Shortcut
--------------------------------
1. Click the three dots menu (top right)
2. Go to: "More tools" → "Create shortcut..."
3. Check: "Open as window"
4. Name it: "POS Inventory System"
5. Click "Create"

STEP 3: Find Shortcut
---------------------
- Shortcut appears on Desktop
- Double-click to open
- Opens in its own window (like an app!)

STEP 4: Add Custom Icon (Optional)
----------------------------------
1. Right-click shortcut → Get Info
2. Click icon in top left
3. Copy your TDO icon (Cmd+C)
4. Click shortcut icon
5. Paste (Cmd+V)

================================================================================
METHOD 3: CREATE ALIAS/SYMLINK (Simple)
================================================================================

STEP 1: Create Alias
--------------------
1. Open Finder
2. Navigate to your project:
   Documents → " Old Laptop Documents" → CAO → FINANCIAL REPORT → pos-inventory-system → frontend → build

3. Right-click "index.html"
4. Select "Make Alias"
5. Drag alias to Desktop
6. Rename to: "POS Inventory System"

STEP 2: Change Icon
-------------------
1. Right-click alias → Get Info
2. Click icon → Copy your TDO icon → Paste

STEP 3: Use It
--------------
- Double-click opens in browser
- Quick access to your app

================================================================================
METHOD 4: USE AUTOMATOR (Advanced - Creates Real App)
================================================================================

STEP 1: Open Automator
----------------------
1. Open "Automator" (Applications → Automator)
2. Choose: "Application"

STEP 2: Create Workflow
-----------------------
1. Drag "Run Shell Script" to workflow
2. Paste this:

   open -a "Google Chrome" "http://localhost:3000"

   OR for Safari:

   open -a "Safari" "http://localhost:3000"

3. File → Save
4. Name: "POS Inventory System"
5. Save to: Desktop

STEP 3: Add Icon
----------------
1. Right-click app → Get Info
2. Click icon → Copy TDO icon → Paste

STEP 4: Use It
--------------
- Double-click opens app in browser
- Looks like a real Mac app!

================================================================================
RECOMMENDED: METHOD 2 (Chrome Create Shortcut)
================================================================================

This is the EASIEST and BEST method:

1. Open Chrome
2. Go to http://localhost:3000
3. Menu → More tools → Create shortcut
4. Check "Open as window"
5. Create
6. Done!

The shortcut will:
- Appear on Desktop
- Open in its own window
- Work like a native app
- Can add your TDO icon

================================================================================
FOR DEPLOYED APP (When Online)
================================================================================

Once you deploy to Netlify/Vercel:

1. Open Chrome
2. Go to your deployed URL (e.g., https://your-app.netlify.app)
3. Menu → More tools → Create shortcut
4. Check "Open as window"
5. Create

Now you have a desktop icon that opens your online app!

================================================================================
ADD YOUR TDO ICON TO DESKTOP SHORTCUT
================================================================================

STEP 1: Prepare Icon
--------------------
1. Open your TDO icon (logo512.png) in Preview
2. Select All (Cmd+A)
3. Copy (Cmd+C)

STEP 2: Apply to Shortcut
-------------------------
1. Right-click desktop shortcut
2. Get Info
3. Click the icon in top left (small icon preview)
4. Paste (Cmd+V)
5. Icon changes to your TDO icon!

================================================================================
QUICK SETUP FOR LOCAL APP
================================================================================

1. Open Chrome
2. Go to: http://localhost:3000
3. Menu (3 dots) → More tools → Create shortcut
4. Name: "POS Inventory System"
5. Check: "Open as window"
6. Click "Create"
7. Desktop icon appears!
8. (Optional) Add TDO icon: Right-click → Get Info → Paste icon

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "Create shortcut" not in menu
---------------------------------------
SOLUTION:
- Make sure you're in Chrome (not Safari/Firefox)
- Try: Menu → More tools → Create shortcut
- Or use Method 1 (Script Editor)

PROBLEM: Shortcut opens but shows error
---------------------------------------
SOLUTION:
- Make sure server is running (npx serve -s build -p 3000)
- Check URL is correct: http://localhost:3000
- Try opening in browser first to verify it works

PROBLEM: Can't change icon
---------------------------
SOLUTION:
- Make sure you copy the icon first (Cmd+C)
- Click the small icon preview in Get Info window
- Then paste (Cmd+V)
- Icon must be PNG or ICNS format

PROBLEM: Shortcut doesn't work
------------------------------
SOLUTION:
- Check server is running
- Verify URL is correct
- Try creating shortcut again
- Make sure Chrome is set as default browser
