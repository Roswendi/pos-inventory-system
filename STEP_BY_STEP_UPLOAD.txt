STEP 7: CONFIGURE APPLICATION
==============================

OPTION A: Using Terminal in hPanel
------------------------------------
1. In hPanel, find: "Terminal" or "SSH"
2. Click on it
3. You'll see a command prompt

Run these commands one by one:

  cd public_html
  npm install --production
  cp .env.example .env
  nano .env

4. In nano editor, add these lines:
   NODE_ENV=production
   PORT=5001
   JWT_SECRET=change-this-to-random-string-12345

5. Save: Press Ctrl+X, then Y, then Enter

6. Set permissions:
   chmod -R 755 .
   chmod -R 777 data
   chmod -R 777 uploads

OPTION B: Contact Hostinger Support
------------------------------------
If you can't access Terminal:

1. Use Live Chat in hPanel
2. Tell them:
   "I uploaded a Node.js app. Can you help me:
   - Run: npm install --production
   - Create .env file with NODE_ENV=production, PORT=5001
   - Set file permissions"

They will do it for you!
