METHOD 1: UPLOAD FILE (Easiest - Copy-Paste or Drag-Drop)

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Copy-Paste doesn't work
----------------------------------
SOLUTION:
- Make sure you're clicking IN the upload area first
- Try drag & drop instead
- Or use Browse Files button
- Check browser console (F12) for errors

PROBLEM: Image doesn't upload
------------------------------
SOLUTION:
- Check file size (must be under 5MB)
- Check file format (must be image)
- Make sure backend is running
- Try different image file
- Check browser console for errors

PROBLEM: Preview doesn't show
------------------------------
SOLUTION:
- Wait a few seconds for upload to complete
- Check if image uploaded successfully
- Try refreshing the form
- Check browser console (F12)

PROBLEM: Image shows broken
----------------------------
SOLUTION:
- Make sure backend is running on port 5001
- Check uploads folder exists: backend/uploads/
- Try uploading again
- Check file format is supported
