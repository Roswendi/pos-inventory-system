METHOD 1: ONLINE TOOL - QUICKEST (5 minutes)
================================================================================

This is the fastest way if you just want text "TDO" as your icon.

STEP 1: Go to Online Icon Generator
-----------------------------------
1. Open your web browser
2. Go to: https://www.favicon-generator.org/
   OR
   Go to: https://favicon.io/favicon-generator/

STEP 2: Create Text Icon
-------------------------
1. Look for "Text to Favicon" or "Text Icon Generator"
2. Enter text: "TDO"
3. Choose font: Bold, Sans-serif (like Arial Bold)
4. Choose background color: #0066FF (blue) or your brand color
5. Choose text color: White (#FFFFFF)
6. Adjust size if needed

STEP 3: Generate and Download
-------------------------------
1. Click "Generate" or "Create"
2. Download the generated package
3. Extract the ZIP file

STEP 4: Find Required Files
-----------------------------
Look for these files in the downloaded package:
- android-chrome-192x192.png
- android-chrome-512x512.png

STEP 5: Rename and Place Files
-------------------------------
1. Rename android-chrome-192x192.png to: logo192.png
2. Rename android-chrome-512x512.png to: logo512.png
3. Copy both files to: frontend/public/

METHOD 2: CANVA.COM - RECOMMENDED (10 minutes)
================================================================================

This method gives you more control and professional results.

STEP 1: Sign Up/Login to Canva
-------------------------------
1. Go to: https://www.canva.com
2. Sign up for free account (or login if you have one)
3. Canva is free to use

STEP 2: Create New Design
---------------------------
1. Click "Create a design" button (top right)
2. Click "Custom size"
3. Enter dimensions:
   - Width: 512 pixels
   - Height: 512 pixels
4. Click "Create new design"

STEP 3: Design Your TDO Icon
-----------------------------
Option A: Simple Text Logo
---------------------------
1. Click "Text" in left sidebar
2. Click "Add a heading" or "Add a subheading"
3. Type: "TDO"
4. Select the text
5. Format text:
   - Font: Choose bold font (Montserrat Bold, Poppins Bold, or Arial Bold)
   - Size: Make it large (around 300-400px)
   - Color: White (#FFFFFF)
   - Center it on the canvas

Option B: Text with Background
-------------------------------
1. Click "Elements" in left sidebar
2. Search for "square" or "circle"
3. Add a square shape
4. Resize to fill most of canvas
5. Change color to: #0066FF (blue) or your brand color
6. Send to back (right-click â†’ "Send to back")
7. Add text "TDO" on top (as in Option A)
8. Make text white

Option C: Text with Decorative Elements
----------------------------------------
1. Add background shape (square or circle)
2. Add text "TDO"
3. Add decorative elements:
   - Lines, borders, or shapes
   - Small icons or symbols
   - Gradients or patterns
4. Keep it simple and clean

STEP 4: Export as 512x512
--------------------------
1. Click "Download" button (top right)
2. Select file type: "PNG"
3. Make sure size is 512x512 (should be automatic)
4. Click "Download"
5. Save file as: logo512.png

6. DESIGN TIPS FOR TDO ICON
================================================================================

COLOR SCHEMES:
--------------
Option 1: Blue Background
- Background: #0066FF (blue)
- Text: #FFFFFF (white)
- Professional and modern

Option 2: Dark Background
- Background: #1A1A1A (dark gray/black)
- Text: #FFFFFF (white)
- Elegant and sophisticated

Option 3: Gradient
- Background: Gradient from #0066FF to #0044CC
- Text: #FFFFFF (white)
- Modern and dynamic

Option 4: White Background
- Background: #FFFFFF (white)
- Text: #0066FF (blue)
- Clean and minimal

FONT RECOMMENDATIONS:
---------------------
- Arial Bold (clean, readable)
- Montserrat Bold (modern)
- Poppins Bold (friendly)
- Roboto Bold (professional)
- Helvetica Bold (classic)

DESIGN STYLES:
--------------
Style 1: Simple Text
- Just "TDO" text
- Bold font
- Centered
- Clean background

Style 2: Text with Border
- "TDO" text
- Border/outline around text
- Adds definition

Style 3: Text with Shape
- "TDO" inside circle or rounded square
- Modern look
- More visual interest

Style 4: Text with Icon
- "TDO" text
- Small icon/symbol (optional)
- More branded look

SIZE GUIDELINES:
----------------
- Text should be 60-80% of icon size
- Leave 10-20% padding around edges
- For 512px: Text around 300-400px
- For 192px: Text around 115-150px

CONTRAST TIPS:
--------------
- High contrast is essential
- Dark background + light text = best
- Test at small size (192px) to ensure readability
- Avoid similar colors (e.g., light blue on white)

TESTING:
--------
- View icon at actual size (192px and 512px)
- Test on different backgrounds
- Check on mobile device
- Ensure text is readable at small size

8. TROUBLESHOOTING
================================================================================

PROBLEM: Icon not showing in browser
-------------------------------------
SOLUTION:
- Check file names are exactly: logo192.png and logo512.png
- Verify files are in frontend/public/ folder
- Clear browser cache
- Rebuild: npm run build
- Check browser console for errors

PROBLEM: Icon looks blurry
---------------------------
SOLUTION:
- Ensure dimensions are exactly 192x192 and 512x512
- Use PNG format (not JPG)
- Don't resize smaller version from larger - create separately
- Use high-quality export settings

PROBLEM: Text "TDO" not readable
---------------------------------
SOLUTION:
- Increase font size
- Use bolder font
- Increase contrast (darker background, lighter text)
- Test at 192px size to ensure readability
- Simplify design

PROBLEM: Wrong colors
----------------------
SOLUTION:
- Use hex color codes (#0066FF for blue, #FFFFFF for white)
- Check color mode (RGB, not CMYK)
- Export as PNG (preserves colors better than JPG)

PROBLEM: File too large
------------------------
SOLUTION:
- PNG files should be 10-50KB for 192px
- PNG files should be 50-200KB for 512px
- If larger, optimize using: https://tinypng.com
- Or reduce complexity of design

PROBLEM: Can't find files after creation
------------------------------------------
SOLUTION:
- Check Downloads folder
- Check Desktop
- Use search: logo192.png or logo512.png
- Remember where you saved them
