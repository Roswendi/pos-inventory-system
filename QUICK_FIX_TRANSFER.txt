================================================================================
    QUICK FIX: TRANSFER TO HOSTINGER - SIMPLEST METHOD
================================================================================

✅ NO FTP NEEDED - USE FILE MANAGER
================================================================================

METHOD 1: UPLOAD SINGLE FILE (EASIEST)
======================================

STEP 1: You already have the file!
-----------------------------------
  hostinger-pos-system.tar.gz

STEP 2: Upload via File Manager
--------------------------------
1. Log into Hostinger hPanel
2. Go to: File Manager
3. Navigate to: public_html
4. Click: Upload button
5. Select: hostinger-pos-system.tar.gz
6. Click: Upload
7. Wait for upload (2-5 minutes)

STEP 3: Extract on Server
--------------------------
1. In File Manager, find: hostinger-pos-system.tar.gz
2. Right-click it
3. Select: Extract
4. Wait for extraction

STEP 4: Configure
----------------
1. In hPanel, go to: Terminal or SSH
2. Run:
   cd public_html
   npm install --production
   cp .env.example .env
   nano .env

3. Add to .env:
   NODE_ENV=production
   PORT=5001
   JWT_SECRET=your-secret-key

4. Save: Ctrl+X, Y, Enter

5. Set permissions:
   chmod -R 755 .
   chmod -R 777 data
   chmod -R 777 uploads

6. Start app (if Node.js App):
   - Go to Node.js section in hPanel
   - Start your app

Done! ✅

IF FILE MANAGER DOESN'T WORK:
==============================

OPTION A: Try FTP with Different Settings
-------------------------------------------
1. Download FileZilla
2. In FileZilla: Edit → Settings
3. Connection → FTP
4. Enable: "Passive mode"
5. Try connecting again

OPTION B: Contact Hostinger Support
------------------------------------
1. Use Live Chat (24/7)
2. Tell them: "I need help uploading Node.js app"
3. They can help you directly

OPTION C: Use Hostinger Node.js App
------------------------------------
1. In hPanel, go to: Node.js section
2. Create new Node.js app
3. Some plans allow direct file upload
4. Follow their interface

COMMON ISSUES:
==============

ISSUE: Upload fails
-------------------
→ Try uploading in smaller batches
→ Check file size (should be < 50MB)
→ Try compressing files first

ISSUE: Files uploaded but not visible
--------------------------------------
→ Refresh File Manager
→ Check you're in correct folder (public_html)
→ Check file permissions

ISSUE: Application doesn't work after upload
--------------------------------------------
→ Check: npm install completed
→ Check: .env file exists
→ Check: Node.js is installed
→ Check server logs

ISSUE: Can't access Terminal/SSH
---------------------------------
→ Contact Hostinger support
→ Ask them to run commands for you
→ Or use their support chat
