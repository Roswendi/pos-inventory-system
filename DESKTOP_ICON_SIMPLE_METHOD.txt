âœ… METHOD 1: Use Automator (Easiest - No Terminal)
================================================================================

STEP 1: Open Automator
-----------------------
1. Press Cmd+Space
2. Type "Automator"
3. Press Enter

STEP 2: Create New Application
------------------------------
1. Choose "Application"
2. Click "Choose"

STEP 3: Add "Run Shell Script" Action
-------------------------------------
1. In the left sidebar, search for "Run Shell Script"
2. Drag it to the workflow area

STEP 4: Add Script
------------------
In the script box, paste this:

  cd "/Users/endiroswendi/Documents/ Old Laptop Documents/CAO/FINANCIAL REPORT/pos-inventory-system/backend"
  
  # Check if server is running
  if ! lsof -ti:5001 > /dev/null 2>&1; then
      # Start server in background
      node server.js > /dev/null 2>&1 &
      sleep 3
  fi
  
  # Open browser
  open "http://localhost:5001"

STEP 5: Save Application
-------------------------
1. File â†’ Save
2. Name: "TDO POS System"
3. Save to: Desktop
4. File Format: Application
5. Click "Save"

STEP 6: Add Icon (Optional)
----------------------------
1. Right-click the app on desktop
2. Select "Get Info"
3. Click the icon in top-left
4. Press Cmd+C to copy an image
5. Click the icon again
6. Press Cmd+V to paste

Done! Double-click the icon to launch!

âœ… METHOD 5: Create .app Bundle Manually
================================================================================

STEP 1: Create Folder Structure
--------------------------------
On Desktop, create:
  TDO POS System.app/Contents/MacOS/

STEP 2: Create Launcher Script
-------------------------------
Create file: TDO POS System.app/Contents/MacOS/launcher

Content:
  #!/bin/bash
  cd "/Users/endiroswendi/Documents/ Old Laptop Documents/CAO/FINANCIAL REPORT/pos-inventory-system/backend"
  if ! lsof -ti:5001 > /dev/null 2>&1; then
      osascript -e 'tell app "Terminal" to do script "cd '"'"'/Users/endiroswendi/Documents/ Old Laptop Documents/CAO/FINANCIAL REPORT/pos-inventory-system/backend'"'"' && node server.js"'
      sleep 5
  fi
  open "http://localhost:5001"

Make executable:
  chmod +x "TDO POS System.app/Contents/MacOS/launcher"

STEP 3: Create Info.plist
--------------------------
Create: TDO POS System.app/Contents/Info.plist

Content:
  <?xml version="1.0" encoding="UTF-8"?>
  <!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
  <plist version="1.0">
  <dict>
      <key>CFBundleExecutable</key>
      <string>launcher</string>
      <key>CFBundleName</key>
      <string>TDO POS System</string>
  </dict>
  </plist>

Done! Double-click to launch.
