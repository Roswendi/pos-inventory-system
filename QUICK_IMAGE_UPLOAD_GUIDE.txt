================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Paste doesn't work
----------------------------
SOLUTION:
- Make sure you click IN the upload area first
- Try drag & drop instead
- Or use Browse Files button

PROBLEM: Image doesn't upload
------------------------------
SOLUTION:
- Check file size (must be under 5MB)
- Check backend is running (port 5001)
- Try different image
- Check browser console (F12)

PROBLEM: Preview doesn't show
------------------------------
SOLUTION:
- Wait a few seconds
- Check if upload completed
- Try refreshing form
